Regra de ouro: O gradiente cyan→green→yellow é a assinatura visual da SeekMedia. Ele aparece na barra inferior de cada slide, nos títulos de destaque e nos elementos interativos. Nunca use o gradiente em texto corrido — só em headings, barras decorativas e CTAs.

2. Gradientes

Gradientes são o DNA visual da marca

Gradiente Primário (Barra de assinatura)

/* Barra inferior que aparece em todas as páginas */
background: linear-gradient(90deg, #00F0D0, #40F0A0, #80F090, #B0F070, #D0F060, #E0F050);

Gradiente Glow (Efeitos de brilho)

/* Usado em hover, sombras e efeitos de luz */
background: linear-gradient(135deg, #10C0B0, #00F0D0, #80F090);

5. Regras Visuais

O que fazer e o que evitar

FAZER
  • Fundo sempre escuro (#101818)
  • Gradiente só em títulos, barras, botões e métricas
  • Fotos com tratamento de cor esverdeado/ciano
  • Espaçamento generoso entre seções
  • Barra gradiente na base de cada seção
  • Números em destaque com peso 800 + gradiente
  • Cards com bordas sutis (#2E3A3A)
  • Texto branco (#F8F8F8) para conteúdo principal
  • Animações suaves (ease, 300-500ms)
EVITAR
  • Fundo branco ou claro em qualquer página
  • Gradiente em texto corrido (só headings)
  • Cores fora da paleta (vermelho, rosa, azul puro)
  • Fotos sem tratamento de cor
  • Texto cinza claro em fundo cinza (baixo contraste)
  • Bordas grossas ou coloridas em cards
  • Mais de 2 gradientes na mesma seção
  • Tipografia serifada
  • Animações bruscas ou exageradas

6. Tratamento de Imagens

Como as fotos aparecem no catálogo

Fotos de Influenciadores

Fotos com overlay de cor ciano/esverdeada sutil. Contorno com glow no estilo "neon" usando a cor #10C0B0 com blur. O efeito visual é como se houvesse uma luz de neon atrás/ao redor da pessoa.

/* Glow effect ao redor de fotos */
box-shadow: 0 0 40px rgba(16, 192, 176, 0.3), 0 0 80px rgba(0, 240, 208, 0.15);

/* Overlay na foto */
filter: saturate(0.8) hue-rotate(-10deg);

Elementos 3D

O catálogo usa elementos 3D com materiais vítreos (glass morphism) em tons de verde/ciano. Esses elementos aparecem na capa e como decoração. Para o site, esse estilo pode ser usado no hero com animação de scroll (conforme o guia do Jack Roberts).

7. CSS Variables (Copiar e Colar)

Pronto para usar no código do site

/* ═══ SeekMedia Brand Colors ═══ */
:root {
  /* Backgrounds */
  --bg-primary: #101818;
  --bg-surface: #1A2228;
  --bg-elevated: #202830;
  --bg-border: #2E3A3A;

  /* Text */
  --text-primary: #F8F8F8;
  --text-muted: #8899A0;

  /* Accent — Gradient stops */
  --accent-cyan: #00F0D0;
  --accent-green: #80F090;
  --accent-lime: #D0F060;
  --accent-yellow: #E0F050;

  /* Support */
  --teal: #10C0B0;
  --deep-green: #10A010;

  /* Gradients */
  --gradient-bar: linear-gradient(90deg, #00F0D0, #40F0A0, #80F090, #B0F070, #D0F060, #E0F050);
  --gradient-glow: linear-gradient(135deg, #10C0B0, #00F0D0, #80F090);
  --gradient-text: linear-gradient(135deg, #00F0D0, #80F090, #E0F050);
}
Como usar este guia: Quando for construir o site com Claude Code no Anti-Gravity, cole o bloco de CSS variables acima como primeira instrução. Diga ao Claude: "Use este brand guide como base visual. Fundo escuro, gradientes neon, estilo da SeekMedia." Isso garante consistência em todas as páginas.
