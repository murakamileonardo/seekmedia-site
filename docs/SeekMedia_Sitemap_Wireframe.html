Home

Navbar
Logo SeekMedia à esquerda. Menu: Sobre, Casting, Cases, Serviços, Contato. Botão CTA "Fale Conosco" em destaque à direita. Menu hambúrguer no celular.
Logo Links de navegação Fale Conosco
Hero Section
Tela cheia com vídeo de fundo ou animação 3D (do Passo 3 do guia). Título grande e impactante tipo: "Conectamos marcas aos maiores criadores do Brasil". Subtítulo curto explicando o que a SeekMedia faz. Dois botões: "Conheça Nosso Casting" e "Quero um Orçamento".
Vídeo/Animação fullscreen Headline Subtítulo Ver Casting Pedir Orçamento
Barra de Logos de Clientes
Carrossel horizontal automático com logos das marcas parceiras. Fundo neutro, logos em escala de cinza que ficam coloridos no hover. Texto acima: "Marcas que confiam na SeekMedia".
Título da seção Carrossel infinito de logos
Destaques do Casting
Grid com 6 a 8 influenciadores em destaque. Cards com foto, nome, nicho e seguidores. Ao clicar, vai para o perfil completo. Botão abaixo: "Ver Casting Completo".
Grid de cards (3-4 por linha) Foto + nome + nicho + seguidores Ver Casting Completo
Números / Social Proof
Contadores animados: X+ influenciadores, Y+ campanhas, Z+ marcas atendidas, W+ milhões de alcance. Animação de contagem quando a seção entra na tela.
4 contadores animados Ícones ou ilustrações
Cases em Destaque
2 a 3 cases principais em cards grandes. Cada card mostra: logo da marca, nome do influenciador, thumbnail do conteúdo e resultado principal (ex: "2M views em 24h"). Link para a página completa do case.
Cards grandes de cases Resultado em destaque Ver Todos os Cases
Serviços Resumo
3 blocos lado a lado: Gestão de Influenciadores (90%), Produção Audiovisual, Estratégia e Planejamento. Ícone, título curto e 1-2 linhas de descrição cada. Link "Saiba mais" para a página de Serviços.
3 blocos de serviço Ícone + título + descrição
CTA Final
Faixa com fundo em gradiente. Texto tipo: "Pronto para levar sua marca ao próximo nível?". Dois caminhos: "Sou uma Marca" e "Sou Influenciador" — cada um leva para um fluxo diferente.
Sou uma Marca Sou Influenciador
Footer
Logo, links rápidos, redes sociais (Instagram, TikTok, LinkedIn, YouTube), email e telefone. Barra de logos de clientes pode se repetir aqui de forma compacta.

Filtros

Categorias (botões/pills) Ordenar por Plataforma Limpar filtros

Cards em grid responsivo (3 colunas no desktop, 2 no tablet, 1 no celular). Cada card mostra: foto de perfil, nome, nicho/categoria, número de seguidores, plataformas ativas (ícones). Hover revela botão "Ver Perfil". Como são até 50, carrega todos de uma vez (sem paginação), usando lazy loading nas imagens para performance.

Performance: Com até 50 influenciadores, os filtros funcionam no próprio navegador (client-side) — sem necessidade de backend pesado. As imagens usam lazy loading e formato WebP para carregar rápido.

Simulador de Campanhas

/simulador
Interativo Lead Generation Educativo
Toggle de Modo
Toggle no topo da página alternando entre dois modos: "Sou uma Marca" e "Sou Influenciador". Visual tipo switch com as duas opções, fundo muda de cor conforme o modo selecionado.
Toggle Marca/Influenciador
Modo Marca — Controles
Painel de controles interativos no lado esquerdo. Parâmetros: Tamanho do influenciador (Dial: Nano a Mega), Quantidade de influenciadores (Dial: 1 a 10+), Plataformas (Toggles: Instagram / TikTok / YouTube), Tipo de conteúdo (Seletor: Stories / Feed / Reels-Short / Vídeo dedicado), Duração da campanha (Dial: 1 semana a 6 meses), Exclusividade (Toggle On/Off), Porte da marca (Seletor: Startup-PME / Média empresa / Grande empresa / Multinacional), Nível de nicho do produto (Dial: Massa a Nichado), Faixa de investimento (Seletor: Explorando / Até R$50K / R$50-200K / R$200K+).
Dial Tamanho Dial Quantidade Toggle Plataformas Seletor Conteúdo Dial Duração Toggle Exclusividade Seletor Porte Dial Nicho Seletor Investimento
Modo Marca — Resultado
Lado direito da tela. Gráfico radar (aranha) com 5 eixos animados: Alcance Potencial, Engajamento Esperado, Durabilidade do Conteúdo, Custo Relativo, Complexidade de Gestão. Os eixos se movem em tempo real conforme o usuário ajusta os controles. Abaixo do gráfico, conclusão dinâmica que muda conforme a complexidade de gestão (de "campanha direta" até "campanha complexa que se beneficia de agência especializada").
Gráfico Radar 5 eixos Conclusão Dinâmica Montar com a Seek
Modo Influenciador — Controles
Parâmetros: Seguidores totais (Dial: 10K a 5M+), Taxa de engajamento (Dial: 1% a 15%), Plataformas ativas (Toggles: Instagram / TikTok / YouTube / Twitter), Frequência de postagem (Dial: 1x semana a diário), Nicho definido (Toggle: Sim / Não / Mais ou menos), Experiência com marcas (Seletor: Nunca / Algumas vezes / Frequente).
Dial Seguidores Dial Engajamento Toggle Plataformas Dial Frequência Toggle Nicho Seletor Experiência
Modo Influenciador — Resultado
Gauge/velocímetro de Score de Atratividade (0-100) + 3 barras horizontais: Potencial de Monetização, Versatilidade, Atratividade para Marcas. Mensagem personalizada por faixa: Score 80-100 (perfil altamente atrativo, CTA para candidatar-se ao casting), Score 50-79 (base sólida com ajustes sugeridos, Seek ajuda a crescer), Score 20-49 (caminho certo, foco em nicho e consistência, Seek orienta), Score 0-19 (todo criador começou de algum lugar, dicas para crescer).
Gauge Score 0-100 3 barras de indicadores Mensagem personalizada CTA Adaptativo
V2 — Inteligência Artificial: Na segunda fase, o simulador ganha uma camada de IA que analisa a empresa/marca automaticamente e sugere o tipo ideal de campanha (Performance, Branding ou Projetos Especiais), o mix de influenciadores e o formato de conteúdo recomendado.

Painel Admin (Cadastro de Influenciadores)

/admin (acesso restrito)
No-Code Painel interno Responsivo
Login Simples
Tela de login com email + senha. Sem necessidade de cadastro público — só usuários criados internamente.
Lista de Influenciadores
Tabela com todos os influenciadores cadastrados. Colunas: foto, nome, categoria, seguidores, status (ativo/inativo). Busca e filtros. Botões de editar e excluir.
Tabela responsiva Buscar + Novo Influenciador
Formulário de Cadastro / Edição
Modal ou página com campos: Nome, @handle, Bio, Foto de perfil, Cover photo, Categoria/Nicho, Plataformas (Instagram/TikTok/YouTube + links), Seguidores por plataforma, Taxa de engajamento, Localização, Marcas parceiras, Portfolio (upload de imagens). Tudo visual, sem código — qualquer funcionário consegue usar.
Dados básicos Upload de fotos Redes sociais + métricas Categorias (multi-select) Portfolio Salvar
Abordagem No-Code recomendada: Os dados dos influenciadores podem ficar em um Google Sheets ou Airtable como "banco de dados", e o painel admin se conecta via API. Assim, se alguém quiser, pode editar direto na planilha também. O site público puxa os dados da mesma fonte.

Stack Técnica Sugerida

Ferramentas recomendadas para construir o site seguindo o método do Jack Roberts

Claude Code + Anti-Gravity
Gera o código do site inteiro com IA. Você descreve, ele constrói.
Next.js ou Astro
Framework que o Claude vai usar para criar o site. Rápido, SEO-friendly e moderno.
Airtable ou Google Sheets
Banco de dados no-code para os influenciadores. Fácil de editar, conecta com o site via API.
Vercel
Hospedagem gratuita. Publica o site com um comando e atualiza automaticamente.
Cloudinary
Armazenamento otimizado de imagens. Converte para WebP automaticamente e redimensiona.
GitHub
Repositório do código. Integra com Vercel para deploy automático.
